# Woolf University Courses

## 05: Relational Databases Concepts and Techniques

This repository is intended for publishing Relational Databases Concepts and Techniques course homeworks

### Homeworks

* **[HW #2: Relational Database Normalization](./rdb-hw-02/README.md)**  
  Step-by-step breakdown of transforming unnormalized data to Third Normal Form (3NF), including ER diagrams and a DDL script.

* **[HW #3: SQL Database Setup and Basic Queries](./rdb-hw-03/README.md)**  
  Translating raw CSV data into a structured relational database schema using DDL. Also covers basic SQL queries like `SELECT`, aggregations, sorting, and grouping.

* **[HW #4: DDL/DML Commands and Complex SQL Queries](./rdb-hw-04/README.md)**  
  Creating a new Library Management schema, inserting sample data, and writing complex `JOIN` queries. Also covers comparing `INNER` and `LEFT` joins, grouping, and pagination.

* **[HW #5: Nested SQL Queries and Code re-usage](./rdb-hw-05/README.md)**  
  Exploring advanced SQL techniques including nested subqueries (in `SELECT`, `WHERE`, `FROM`), Common Table Expressions (CTEs) with `WITH`, and creating custom user-defined functions.

* **[HW #7: Working with Time and Additional built-in functions](./rdb-hw-07/README.md)**  
  Manipulating date and time attributes using `EXTRACT`, `DATE_ADD`, and `UNIX_TIMESTAMP`, as well as formatting relational data into JSON using `JSON_OBJECT`.

  
* **[Final Project: Relational Databases](./rdb-fp/README.md)**  
  A comprehensive project covering the full database lifecycle: schema design, data normalization to 3NF, complex data analysis, and the implementation of custom SQL functions.
